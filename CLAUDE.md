# Form Dropoff Tracker — Project Context

Analytics dashboard for tracking onboarding form drop-off on whatifweb.co.nz. Mounted at `/tracker` via Webflow Cloud.

## Stack
- **Framework:** Astro (SSR) with `@astrojs/cloudflare` adapter
- **Hosting:** Webflow Cloud (Cloudflare Workers), auto-deploys on push to `main`
- **Database:** Cloudflare D1 (SQLite) via Drizzle ORM — binding name `DB`
- **Styles:** What IF Web v2 design system via DevLink (`devlink/global.css`)
- **Charts:** Chart.js 4 + chartjs-plugin-datalabels (CDN, `is:inline`)

## Key files
- `src/pages/index.astro` — dashboard UI + all client-side JS
- `src/pages/api/track.ts` — POST endpoint, receives events from `tracker.js`
- `src/pages/api/data.ts` — GET endpoint, returns all events as JSON
- `public/tracker.js` — embed script on whatifweb.co.nz onboarding form
- `src/db/schema.ts` — Drizzle schema for D1
- `drizzle/` — SQL migration files
- `devlink/global.css` — generated by DevLink sync (gitignored, regenerate with `webflow devlink sync --site 6719702b25f42f173940b521`)

## Local development
```bash
npm run build && npx wrangler pages dev dist
```
Open at `http://127.0.0.1:8788/tracker`.

For hot-reload layout/style work (no D1): `npm run dev`.

## Design system
- Webflow site ID: `6719702b25f42f173940b521`
- Use Webflow utility classes wherever possible — avoid writing custom CSS for things already covered by the design system
- Key classes: `heading-style-h2-small`, `heading-style-h3`, `text-size-large`, `text-size-tiny`, `text-weight-semibold`, `text-color-secondary`, `text-style-allcaps`
- CSS variables: `--background-color--background-primary/secondary`, `--text-color--text-primary/secondary`, `--border-color--border-tertiary`, `--base-color-brand--blue`, etc.

```html

  <h2 class="heading-style-h3">Label Text</h2>
  <p class="text-size-large" id="statId">0</p>

```
Label is the `<h2>`, value is the `<p>` — not the other way around.
